Get Started

Built with Bootstrap 5, Java and the Streaming Availability API.

The Mariens movie recommendation system uses the Streaming Availability API and our in-house developed algorithm in order to provide you with a movie suitable to your liking. You can start right away by going to our home page and clicking on the Get Started Button. The system is free, but is currently limited to 100 request per day.
